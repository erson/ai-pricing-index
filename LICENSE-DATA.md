## Notes on the underlying prices

Individual prices are facts published by each provider; the `sourceUrl` on every
row points to the provider page the number was read from. ByteCosts does not
claim ownership of the underlying prices — the CC-BY license covers the
**compilation, normalization, schema, and provenance metadata** in this index.

Example code under `examples/` is MIT-licensed (see `LICENSE`).

Prices change. Always confirm against each row's `sourceUrl` before making
billing decisions.
